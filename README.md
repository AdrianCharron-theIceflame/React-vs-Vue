# React vs Vue

This is a small example of using React and Vue 3 to complete the walkthrough given in class

## React App

The React app was created using Vite and given TypeScript support for type security. React-DOM-Router was added later for routing

## Vue 3 App

The Vue 3 app was created with VueRouter and, in this case, showcases both

1. The Composition API (`App.vue` with the `<script setup>` tag), and
2. The Options API (`HomeView.vue` and `UserView.vue` with the `export default` keywords)
